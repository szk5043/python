#### 常用系统模块：

- time
  - 时间戳(timestamp)：time.time()
- datetime
  - 当前时间：datetime.datetime.now()
- calendar
- sys
  - 格式化传参：sys.argv
  - 环境变量：sys.path
- os
  - 指定路径是否存在：os.path.exists(path)
  - 生成单级目录：os.mkdir('dirname')

#### 序列化模块
  -  Pickle
  -  JSON 
    - json.load是从json文件读取数据
    - json.dump是将python数据保存成json文件

#### 随机数和加密：

- random
  - 生成验证码
- md5
  - 获取用户密码哈希值
